1. Frontend

1.1 Culori și font

Paleta de culori folosește verde (#4caf50, #388e3c), albastru (#1976d2) și gri deschis (#e0e7ef) pentru un aspect modern și prietenos. Fontul principal este Arial, ales pentru claritate și lizibilitate pe orice dispozitiv.

1.2 Header & Navigație

Header-ul conține butoane pentru autentificare, logout, acces admin și istoric. Navigarea este intuitivă, cu butoane vizibile și responsive.

1.3 Homepage

Pagina principală permite alegerea tipului de input (vector, matrice, șir, arbore, graf) și afișează formulare dedicate pentru fiecare tip. Aplicația este concepută pentru a genera automat date de test pentru probleme de programare, utile în concursuri sau laboratoare.

1.4 Formulare de generare input

Fiecare tip de input are un formular modern, cu validare și feedback vizual. Formularele sunt responsive și folosesc accente de verde/albastru pentru butoane și highlight-uri. Se pot genera inputuri aleatorii sau personalizate pentru structuri precum array-uri, matrice, grafuri neorientate/orientate și arbori binari.

1.5 Responsive design

Toate componentele sunt optimizate pentru mobil, desktop si tableta, cu layout-uri flexibile și butoane mari, ușor de folosit pe touchscreen.

1.6 Admin Dashboard

Adminul are acces la un panou special pentru gestionarea utilizatorilor: vizualizare, ștergere. Accesul este securizat și disponibil doar pentru utilizatorii cu rol de admin.

2. Funcționalități principale

2.1 Autentificare & Înregistrare

Utilizatorii pot crea cont, se pot autentifica și deconecta. Parolele sunt validate și criptate, iar autentificarea se face cu token JWT.

2.2 Istoric generări

Utilizatorii pot vedea istoricul inputurilor generate, cu detalii despre parametri și data generării.

2.3 Funcții admin

Adminul poate gestiona utilizatori (ștergere, promovare), vedea statistici și accesa funcții suplimentare de moderare.

3. Backend

3.1 Arhitectură & rute

Backend-ul este organizat modular, cu rute pentru autentificare, gestionare utilizatori, generare inputuri și funcții admin. Fiecare rută validează datele și verifică permisiunile.

3.2 Baza de date

Baza de date conține tabele pentru utilizatori, inputuri generate și istoric. Parolele sunt stocate criptat, iar rolurile utilizatorilor sunt gestionate separat.

3.3 API & securitate

Toate endpoint-urile API validează tokenul JWT. Operațiile sensibile (ex: ștergere user) sunt permise doar pentru admini. Datele sunt validate atât pe frontend, cât și pe backend.

3.4 Tehnologii folosite

  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • Backend: Node.js, JWT, bcrypt
  • Baza de date: Oracle

4. Concluzii

Aplicația InputGenerator oferă o experiență modernă și sigură pentru generarea de date de test, cu funcții avansate pentru utilizatori și administratori. Testele de performanță și securitate confirmă fiabilitatea soluției. Este un instrument ideal pentru programatori care au nevoie de inputuri complexe pentru testarea algoritmilor.
